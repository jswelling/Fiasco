Fiasco Parts Info

Some Fiasco shell scripts:

afni_estireg3d.csh
may be substituted for estireg3d.csh to use AFNI's 3dvolreg.
afni_tshift.py
uses the AFNI tool 3dTshift to perform slice timing correction.
align_by_permutation.py
changes a scan's data order to match a prototype, for example converting saggital scans to axial layout.
build_model_matrix.py
uses R to produce a design matrix given a formula and a split file.
calc_stats_over_roi.py
produces a 5-number summary of selected voxels within an ROI
color_by_phase.csh
produces a single image from a complex Pgh MRI dataset, showing magnitude and phase as brightness and color.
combine.csh
combines Tmaps to make group Pmaps using any of several methods.
convert_ifiles.py
converts a directory full of GE image file subdirectories into a collection of Pgh MRI datasets.
coregister.py
implements automatic coregistration of functional and structural scans.
coregister_inplane.py
is used to translate and rotate functional data to align with inplane structural scans.
coregister_makeps.py
is used to generate an overlaid image useful in checking that coregistration was successful.
coregister_struct_to_inplane.py
is used to translate and rotate axial structural data to align with inplane structural scans.
cowarp_inplane.py
is used to estimate a warp for structural scans to compensate for distortion in functional data.
customize_local.py
is a project-specific script for setting up the .local.csh files for individual Fiasco runs.
dataset_matches_env.py
checks that a dataset is compatible with the Fiasco environment variables.
epoch_subset.py
uses a text table of block offsets to select blocks from a file
FIASCO
is the master script which controls Fiasco processing of individual fMRI datasets.
false_discovery.py
calculates P score cutoffs for a given Fiasco Tmap for a specific false discovery rate.
groupcompare.csh
performs comparisons between the response of two different groups to experimental conditions.
icbm_to_tlrc.py
Warps and clips data in the MNI-standard ICBM version of Talairach space to roughly match the AFNI version of Talairach space.
make_cine.csh
produces an MPEG animation from a Pgh MRI dataset.
make_group_anat.csh
produces group averages of structural anatomy datasets.
make_P_P_plots.csh
produces a P-P plot from one or more Pmaps.
make_phase_cine.csh
produces an MPEG animation from a complex Pgh MRI dataset, showing magnitude and phase as brightness and color.
merge_fisher.csh
combines multiple Fiasco session results for the same subject using 3D alignment and Fisher's Method.
merge_detrend.csh
combines multiple Fiasco session results for the same subject by aligning and concatenating the detrended session data.
multi_runner.py
automatically runs batches of individual Fiasco runs.
paste_split.py
concatenates multiple split files
pca.py
performs principle value decomposition.
physio_correct_triggered.py
performs physio correction based on scanner trigger pulse data.
pick_n_voxels.py
Finds threhsolds to select the N most active voxels in a Pmap or Tmap.
plot_roi_event_response.py
plots the response of the voxels within an ROI to particular stimulus events.
plot_roi_time_series.py
plots the time series of a collection of voxels selected by an ROI mask.
pooled_mean.csh
constructs the pooled mean of a group of means with associated counts.
pooled_stdv.csh
constructs the pooled standard deviation of a group of stdvs with associated counts.
pullback_roi_mask.py
pulls a binary mask in Talairach space back into the original head space of a specific subject.
renumber_ifiles.py
builds renumbered links to GE image file directories
stats_to_floats.csh
Generated single precision floating point versions of stats files.
strip_skull.py
strips skulls from anatomical datasets.
testingroup.csh
constructs T and P scores for response to a two-condition test for multiple subjects, all of whom are in the same group.
tlrc_to_icbm.py
Warps and clips data in the AFNI version of Talairach space to roughly match the MNI-standard ICBM of Talairach space.

Fiasco's internal libraries

  • libacct.a and acct.h provide very simple elapsed time accounting functions, enabled with compile time switch -DACCT .
  • libarray.a and array.h provide utility functions that allocate and deallocate 2D and 3D arrays.
  • libbio.a and bio.h provide binary IO utility functions.
  • libcdf.a and cdflib.h contain cumulative statistical distribution functions.
  • libcrg.a and stdcrg.h contain a variety of routines by Chris Genovese, including help function interface.
  • libfmri.a and fmri.h provide random utilities- Fourier transforms, complex types, matrix operations. Also included are a general linear model package (in glm.c), a smoothing package (in smoother.c), and a quaternions package (in quaternion.c).
  • libmdgb.a and mdbg.h look like memory management debugging functions- debug malloc, etc.
  • libmisc.a and misc.h are miscellaneous utility functions by Greg Hood- filename, environment, error message facilities.
  • libmri.a and mri.h contain routines for reading and writing Pittsburgh MRI format files.
  • libpar.a and (npar.h or par.h) contain parallel task distribution routines by Greg Hood.
