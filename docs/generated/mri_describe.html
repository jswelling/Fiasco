mri_describe_help

Introduction


  mri_describe prints short text descriptions of one or more
  Pittsburgh MRI datasets.


  To run mri_describe use:
    mri_describe [-verbose] [-debug] file1 [file2 [file3 ...]]

  or:
    mri_describe -help

Arguments

verbose

  [-verbose]                            (-v)

     Specifies verbose output.  This produces some additional information.

debug

  [-debug]

     Specifies debugging output.

fileN


     Names of Pgh MRI files to be described.
